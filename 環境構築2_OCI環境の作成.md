## 仮想クラウドネットワークの作製
ホーム画面の左ペインより「ネットワーキング」→「仮想クラウド・ネットワーク」を押下する<br>

![alt text](</img/スクリーンショット 2026-02-13 215150.png>)


以下の通り設定して作成する

| 項目 | 入力方法 |
|:---------|:---------|
| 名前 |ネットワーク名 |
| コンパートメントメントに作成 | デフォルト値 |
| IPv4 CIDRブロック | Azureと重複しないIP範囲 |

![alt text](</img/スクリーンショット 2026-02-13 215441.png>)<br>

<br>

## サブネットの作成

ネットワークの作成後、ネットワークの画面が表示されるため、「サブネット」タブ→「サブネットの作成」を押下する

![alt text](</img/スクリーンショット 2026-02-13 215847.png>)<br>

<br>

以下の通り設定して作成する

| 項目 | 入力方法 |
|:---------|:---------|
| 名前 |サブネット名 |
| コンパートメントメントに作成 | デフォルト値 |
| サブネットタイプ | リージョナル |
| IPv4 CIDRブロック | VCNの中から必要なIP範囲 |
| サブネットアクセス | パブリック・サブネット(RDPを行うため) |
| その他 | デフォルト値 |

![alt text](</img/スクリーンショット 2026-02-13 220101.png>)<br>

<br>

## コンピュート インスタンスの作成
ホーム画面の左ペインより「コンピュート」→「インスタンス」を押下する

![alt text](</img/スクリーンショット 2026-02-13 212748.png>)<br>

<br>

基本情報タブにて、以下の通り入力する

| 項目 | 入力方法 |
|:---------|:---------|
| 名前 |インスタンス名 |
| コンパートメントメントに作成 | デフォルト値 |
| オペレーティング・システム | Windows Server 2022 Standard |

![alt text](</img/スクリーンショット 2026-02-13 213717.png>)<br>

<br>

ネットワーキングタブにて、以下の通り入力し、作成する

| 項目 | 入力方法 |
|:---------|:---------|
| VNIC名 | 任意のVNIC名 |
| プライマリ・ネットワーク | 既存の仮想クラウド・ネットワーク |
| 仮想クラウド・ネットワーク | 作製したネットワークを選択 |
| サブネット | 既存のサブネットを選択 |
| サブネット | 作製したサブネット |

![alt text](</img/スクリーンショット 2026-02-13 220913.png>)<br>

<br>

作製したインスタンスについて、インスタンスアクセスからユーザー名と初期パスワードを控える<br>
![alt text](</img/スクリーンショット 2026-02-13 221441.png>)<br>

<br>

## ネットワーク・セキュリティ・グループの設定
ネットワークの画面が表示にて、「セキュリティ」タブ→「ネットワーク・セキュリティ・グループの作成」を押下する

![alt text](</img/スクリーンショット 2026-02-13 222123.png>)<br>

<br>

以下の通り、イングレスとエグレスのルールをすべて許可したルールを作成する

![alt text](</img/スクリーンショット 2026-02-13 223959.png>)<br>

<br>

作製したインスタンスにて、パブリックIPとユーザー名、初期パスワードでローカルPCからログインする<br>
※初期パスワードは変更を求められるため、変更してください。<br>

## DRGの作成
ホーム画面の左ペインより「ネットワーキング」→「動的ルーティング・ゲートウェイ」を押下する

![alt text](</img/スクリーンショット 2026-02-14 065333.png>)<br>

<br>

「動的ルーティング・ゲートウェイ」の作成画面にて、の名前を入力して作成する<br>

<br>

作製したゲートウェイにて、「仮想クラウド・ネットワーク・アタッチメント」の作成を押下する

![alt text](</img/スクリーンショット 2026-02-13 225234.png>)<br>

<br>

以下の通り入力して作成する

| 項目 | 入力方法 |
|:---------|:---------|
| 名前 | 任意のアタッチメント名 |
| 仮想クラウド・ネットワーク | 作製したネットワークを選択 |
| その他 | デフォルト値 |

![alt text](</img/スクリーンショット 2026-02-13 225432.png>)<br>

<br>

<br>

ルート・ルールの追加を押下する

<br>

AzureのIP範囲に対して、作製したDRGへネクストホップが向くようにして作成する
